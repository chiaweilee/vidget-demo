## Vidget Demo

[![Greenkeeper badge](https://badges.greenkeeper.io/chiaweilee/vidget-demo.svg)](https://greenkeeper.io/)

### Install

```
npm install
```

### Build

```
npm run build
```

### Usage

```html
<div id="widget"></div>
<script src="./dist/widget.js"></script>
<script>
window.vidget('#widget', {
    // options
})
</script>
```
